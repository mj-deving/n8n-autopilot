Gelernte Patterns

Dispatcher + Switch

AI klassifiziert Eingabe, Switch routet zu spezialisierten Branches.

chainLlm + Code Parse

LLM-Antwort als Text → Code Node extrahiert JSON → strukturiert weiter.

.uses() fuer AI Sub-Nodes

LLM, Memory und Tools IMMER via .uses(), nie .out().to().

$('Node') Referenz

Daten von frueheren Nodes ueber Branches hinweg mit $('Node Name').item.json.

Webhook + responseNode

Async-Verarbeitung mit synchroner HTTP-Antwort am Ende des Flows.

Conditional Telegram

If-Node fuer bedingte Benachrichtigungen basierend auf Scores/Dringlichkeit.

Gotchas & Troubleshooting

Gemini + Tools Alle Tools brauchen type: OBJECT Schema. Built-in toolThink scheitert — toolCode mit inputSchema nutzen.
n8nac push NUR Dateiname, KEIN Pfad. npx n8nac push 01-ai-news-kurator.workflow.ts
Webhook nach Push Deactivate → Reactivate noetig damit Webhook-ID registriert wird.
Regex in jsCode TS → JSON → JS Escaping-Kette unzuverlaessig. indexOf/lastIndexOf statt Regex fuer JSON-Extraktion.
Telegram Markdown Abgeschnittenes ** crasht. LLM-Output strippen: .split("*").join("")
Gemini Rate Limit Free Tier = 15 req/min. Multi-Agent WFs: Tests mindestens 8-10s auseinander.
