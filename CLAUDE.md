```bash

  http://localhost:5678/api/v1/workflows/<id>/activate -d '{}'
```

## Critical Gotchas

### n8nac
- **Push filename**: ONLY filename, NO path — n8nac resolves from config
- **No constants**: Cannot use `const` variables as node parameter values — only inline literals
- **alwaysOutputData**: Not supported in decorator — must patch via n8n API after each push
- **Conflict resolution**: After API patches, n8nac sees a conflict on next push → `resolve --mode keep-current`

### n8n
- **WEBHOOK_URL**: The env var is `WEBHOOK_URL`, NOT `N8N_WEBHOOK_URL` (no prefix!)
- **Windows env vars**: `VAR=x cmd &` doesn't propagate in Git Bash. Use PowerShell: `$env:VAR='x'; n8n start`
- **API activation**: Requires `Content-Type: application/json` header + body `{}`
- **n8n API key**: Stored in SQLite `~/.n8n/database.sqlite`, table `user_api_keys`

### AI / Gemini
- **AI sub-nodes**: MUST use `.uses()` not `.out().to()` — `ai_tool` is ARRAY, rest is single ref
- **Gemini + Tools**: All tools need `type: OBJECT` parameter schema. Built-in `toolThink` fails — use Code Tool with `specifyInputSchema: true` + `inputSchema`
- **Gemini tool-calling**: Weaker than Claude at proactive tool use — add explicit instructions in system prompt
- **Rate limit**: Free tier = 15 req/min. Space tests 8+ seconds apart.
- **Telegram markdown crash**: Strip `*`, `` ` ``, `_` from LLM output before sending

### Windows / Process Safety
- **NEVER** use `pkill -f "n8n"` — kills Claude Code too (working dir contains "n8n")
- Kill n8n by specific PID: find port 5678 holder via `powershell Get-NetTCPConnection -LocalPort 5678`, then `taskkill //PID <pid> //F`
- `npx n8n` → ECOMPROMISED on Windows, always use `npm install -g n8n`

### WF6 Post-Push Patch Required
After every `n8nac push`, run this to fix `alwaysOutputData`:
```python
# Patch via API — n8nac doesn't support alwaysOutputData
import json, urllib.request
API_KEY = "<key>"
url = "http://localhost:5678/api/v1/workflows/aRKiZkBhBDytjO1m"
req = urllib.request.Request(url, headers={"X-N8N-API-KEY": API_KEY})
with urllib.request.urlopen(req) as resp:
    wf = json.loads(resp.read())
for node in wf["nodes"]:
    if "Get Init Profile" in node["name"]:
        node["alwaysOutputData"] = True
        node["parameters"].pop("filters", None)
payload = {"name": wf["name"], "nodes": wf["nodes"], "connections": wf["connections"], "settings": wf.get("settings", {})}
req = urllib.request.Request(url, data=json.dumps(payload).encode(), method="PUT",
    headers={"X-N8N-API-KEY": API_KEY, "Content-Type": "application/json"})
urllib.request.urlopen(req)
```
